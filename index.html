Lenguajes de Programacion Aprendidos

HTML

HTML, con significado en ingles HyperText Markup Language, hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

Java Script

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

CSS

Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​

PHP

Es un lenguaje de programación interpretado del lado del servidor y de uso general que se adapta especialmente al desarrollo web.

Phyton

Es un lenguaje de alto nivel de programación. Se trata de un multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y, en menor medida, programación funcional.

SQL

Es un lenguaje específico de dominio, diseñado para administrar, y recuperar información de sistemas de gestión de bases de datos relacionales.

C++

Es un lenguaje de programación cuya intención de su creación fue extender al lenguaje de programación C y añadir mecanismos que permiten la manipulación de objetos.
